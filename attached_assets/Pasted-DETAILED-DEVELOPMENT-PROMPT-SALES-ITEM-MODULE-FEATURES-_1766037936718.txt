1️⃣ Feature – Sales → Items → List Page Filtering
Objective: Add filter functionality on the All Items screen.

Add filter dropdown with the following three filter options:
All Items
Active Items
Inactive Items

System must fetch & refresh item list accordingly.
Default selection = All Items.

GET /api/sales/items?status=all GET /api/sales/items?status=activeGET /api/sales/items?status=inactive

Highlight currently selected filter.
If list empty → show “No items found”.
